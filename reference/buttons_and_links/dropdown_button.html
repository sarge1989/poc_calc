Dropdown button

You can include the following attributes when using the component. Note that those marked with * are compulsory:
  • data-include*: Name of html file of component
  • data-id*: ID of component when inserting to html
  • data-name*: Name of component for retrieving form data
  • data-render: Set value to "true" if the component requires rendering. Most component requires rendering to set the class.
  • data-placeholder*: Placeholder text.
  • data-options*: Options to be included. Format is e.g. {"value":"value1","text":"text1"}|{"value":"value2","text":"text2"}|{"value":"value3","text":"text3"}. This will be split by "|" using js.
  • data-class*: Classes are used to render the different variants. Available class names are:
    • Size*: [Medium] cmp-button-dropdown-md, [Large] cmp-button-dropdown-lg
    • Style*: [Primary] cmp-button-dropdown--primary, [Secondary] cmp-button-dropdown--secondary
    • Theme (no class is Light): [Dark] cmp-button-dropdown--inverse
    • Is disabled: cmp-button--disabled
Examples:

<!-- Primary medium -->

<div data-include="dropdown_button" data-id="dropdown_1" data-name="dropdown-btn1" data-class="cmp-button-dropdown-md cmp-button-dropdown--primary" data-placeholder="Primary medium" data-render="true" data-options='{"value":"option-1","text":"Option 1"}|{"value": "option-2","text":"Option 2"}|{"value": "option-3","text":"Option 3"}'>

<!-- Secondary large - dark -->

<div data-include="dropdown_button" data-id="dropdown_2" data-name="dropdown-btn2" data-class="cmp-button-dropdown-lg cmp-button-dropdown--secondary cmp-button-dropdown--inverse" data-placeholder="Secondary large dark" data-render="true" data-options='{"value":"option-1","text":"Option 1"}|{"value": "option-2","text":"Option 2"}|{"value": "option-3","text":"Option 3"}'> </div>

<!-- Primary large - disabled -->

<div data-include="dropdown_button" data-id="dropdown_3" data-name="dropdown-btn3" data-class="cmp-button-dropdown-lg cmp-button-dropdown--primary cmp-button--disabled" data-placeholder="Primary large" data-render="true" data-options='{"value":"option-1","text":"Option 1"}|{"value": "option-2","text":"Option 2"}|{"value": "option-3","text":"Option 3"}'> </div>
