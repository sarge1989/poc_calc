Standard button

You can include the following attributes when using the component. Note that those marked with * are compulsory:
  • data-include*: Name of html file of component
  • data-id*: ID of component when inserting to html
  • data-render: Set value to "true" if the component requires rendering. Most component requires rendering to set the class.
  • data-btntext: Button text. Exclude this attribute to use component as an icon only button.
  • data-include-icon: Set value to "true" if component includes right icon. Exclude this attribute to use component as an text only button.
  • data-href: Destination URL. Exclude this attribute to use component as a action button (e.g. submit, next, back).
  • data-class*: Classes are used to render the different variants. Available class names are:
    • Size (no class is Large): [Medium] cmp-button--medium
    • Style*: [Primary] cmp-button--primary, [Secondary] cmp-button--secondary, [Tertiary] cmp-button--tertiary
    • Theme (no class is Light): [Dark] cmp-button--inverse
    • Is disabled: cmp-button--disabled
Examples:

<!-- Button using <a> -->

<div data-include="standard_button" data-class="cmp-button--primary cmp-button--medium" data-btntext="Primary medium using <a>" data-id="btn_1" data-render="true" data-include-icon="true" data-href="#"></div>

<!-- Button using <button> -->

<div data-include="standard_button" data-class="cmp-button--primary cmp-button--medium" data-btntext="Primary medium" data-id="btn_2" data-render="true" data-include-icon="true"></div>

<!-- Button without text -->

<div data-include="standard_button" data-class="cmp-button--primary cmp-button--medium" data-id="btn_3" data-render="true" data-include-icon="true"></div>
