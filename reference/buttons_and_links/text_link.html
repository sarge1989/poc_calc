Text link

You can include the following attributes when using the component. Note that those marked with * are compulsory:
  • data-include*: Name of html file of component
  • data-id*: ID of component when inserting to html
  • data-render: Set value to "true" if the component requires rendering. Most component requires rendering to set the class.
  • data-linktext: Link text.
  • data-href: Destination URL.
  • data-iconleft: Set value to "true" if icon is on the left. Else, it will be on the right by default.
  • data-class: Classes are used to render the different variants. Separate each class name with a space. Available class names are:
    • Theme (no class is Light): [Dark] cmp-button--inverse
    • Is disabled: cmp-button--disabled
Examples:

<!-- Text link - right icon -- >

<div data-include="text_link" data-id="tl_1" data-render="true" data-linktext="Text link" data-href="#"></div>

<!-- Text link - left icon -->

<div data-include="text_link" data-id="tl_2" data-render="true" data-linktext="Text link" data-href="#" data-iconleft="true"></div>

<!--Text link - right icon (dark) -->

<div data-include="text_link" data-id="tl_3" data-render="true" data-linktext="Text link" data-href="#"data-class="cmp-button--inverse"></div>
