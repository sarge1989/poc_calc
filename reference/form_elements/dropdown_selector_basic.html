Dropdown selector

You can include the following attributes when using the component. Note that those marked with * are compulsory:
  • data-include*: Name of html file of component
  • data-id*: ID of component when inserting to html
  • data-render: Set value to "true" if the component requires rendering. Most component requires rendering to set the class.
  • data-field-id*: ID of input field (note: this is different from data-id)
  • data-name*: Name of component for retrieving form data
  • data-input-label*: Input field label
  • data-tooltip-content: Tooltip content. Exclude this attribute if tooltip is not required.
  • data-caption*: Caption for displaying help text or error messages
  • data-placeholder*: Placeholder text
  • data-options*: Options to be included. Format is e.g. {"value":"value1","text":"text1"}|{"value":"value2","text":"text2"}|{"value":"value3","text":"text3"}. This will be split by "|" using js.
  • data-default-value*: Default value selected.
Examples:

<!-- Basic -->

<div data-include="dropdown_selector_basic" data-id="dsb_1" data-render="true" data-name="dropdown_selector_basic_1" data-input-label="Dropdown selector - basic" data-field-id="dropdown_selector_basic_1" data-tooltip-content="Some sample content in the tooltip for basic dropdown selector." data-caption="This is a help text for basic dropdown selector." data-options='{"value":"option-1","text":"Option 1"}|{"value": "option-2","text":"Option 2"}|{"value": "option-3","text":"Option 3"}'> </div>
