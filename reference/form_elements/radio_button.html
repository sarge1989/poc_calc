Radio button field

You can include the following attributes when using the component. Note that those marked with * are compulsory:
  • data-include*: Name of html file of component
  • data-id*: ID of component when inserting to html
  • data-render: Set value to "true" if the component requires rendering. Most component requires rendering to set the class.
  • data-name*: Name of component for retrieving form data
  • data-input-label*: Input field label
  • data-options*: Options to be included. Format is e.g. {"value":"value1","text":"text1","selected":true,"caption":"caption1"}|{"value":"value2","text":"text2","caption":"caption2"}|{"value":"value3","text":"text3","caption":"caption3"}. This will be split by "|" using js.
  • data-tooltip-content: Tooltip content. Exclude this attribute if tooltip is not required.
  • data-is-horizontal-stack*: Set value to "true" if the radio button should be shown side-by-side. Exclude this if the radio buttons stack vertically.
  • data-caption*: Caption for displaying help text or error messages
  • data-is-disabled*: Set value to "true" if the component is disabled. Exclude this if component is not disabled.
JS

Include the codes in bold in index.js when using this component. 

import { selectionChangeRadioButton } from "../components/form-elements/selectionChangeRadioButton.js";

// Example
const init = function () {
    $.fn.globalHeader();
    $.fn.DropDownMenu();

    $("input[type='radio']").on('change', selectionChangeRadioButton); // Add this line in init()
};
