Single date picker

You can include the following attributes when using the component. Note that those marked with * are compulsory:
  • data-include*: Name of html file of component
  • data-id*: ID of component when inserting to html
  • data-render: Set value to "true" if the component requires rendering. Most component requires rendering to set the class.
  • data-field-id*: ID of input field (note: this is different from data-id)
  • data-input-label*: Input field label
  • data-tooltip-content: Tooltip content. Exclude this attribute if tooltip is not required.
  • data-caption*: Caption for displaying help text or error messages
  • data-datepicker-type*: Can be "day", "month" or "year"
  • data-default: Default date. Set value to "today" to use today's date as default. Leave empty if no default date is required.
  • data-check-age: Set value to "true" if requirement is to check for age.
  • data-min-age: Minimum age.
  • data-max-age: Minimum age.
  • data-min-date: Minimum date. Applicable only when not checking for age.
  • data-max-date: Minimum date. Applicable only when not checking for age.
JS

Include the codes in bold in index.js when using this component. 

import "../components/datepicker/datepicker-lib.js"; // mandatory import for any type of datepicker
import "../components/datepicker/datepicker-day.js"; // import only if needed
import "../components/datepicker/datepicker-month.js"; // import only if needed
import "../components/datepicker/datepicker-year.js"; // import only if needed

// Example
const init = function () {
    $.fn.globalHeader();
    $.fn.DropDownMenu();

    // For Day picker - Add this line in init() if required
    $(".sb-datepicker-container.cmp-inputs.sb-datepicker-container--day").DatepickerDate();

    // For Month picker - Add this line in init() if required
    $(".sb-datepicker-container.cmp-inputs.sb-datepicker-container--month").DatepickerMonth();

    // For Year picker - Add this line in init() if required
    $(".sb-datepicker-container.cmp-inputs.sb-datepicker-container--year").DatepickerYear(); 
};
