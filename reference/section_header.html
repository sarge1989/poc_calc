Section header

You can include the following attributes when using the component. Note that those marked with * are compulsory:
  • data-include*: Name of html file of component
  • data-id*: ID of component when inserting to html
  • data-render: Set value to "true" if the component requires rendering. Most component requires rendering to set the class.
  • data-overline: Overline text
  • data-headline-style:Can be H2 - H5 (default is H3)
  • data-headline: Headline text
  • data-headline-para: Paragraph under Headline
  • data-subsection-headline: Subsection headline text
  • data-subsection-headline-para: Paragraph under Subsection headline
  • data-linktext: Link text.
  • data-href: Destination URL.
  • data-class: Classes are used to render the different variants. Separate each class name with a space. Available class names are:
    • Theme (no class is Light): [Dark] dark-mode
Examples:

<!-- Section header with all fields -->

<div data-include="section_header" data-id="sh_1" data-render="true" data-overline="This is overline" data-headline-style="H2" data-headline="Headline 2 style" data-headline-para="Displaying section header with H2 style headline." data-subsection-headline="This is a subsection headline." data-subsection-headline-para="This is the subsection paragraph which usually only appears with the subsection headline." data-linktext="Text link" data-href="#"> </div>

<!-- Section header using headline -->

<div data-include="section_header" data-id="sh_2" data-render="true" data-overline="This is overline" data-headline-style="H2" data-headline="Headline 2 style" data-headline-para="Displaying section header with H2 style headline."> </div>

<!-- Section header using subsection headline -->

<div data-include="section_header" data-id="sh_3" data-render="true" data-subsection-headline="This is a subsection headline." data-subsection-headline-para="This is the subsection paragraph which usually only appears with the subsection headline." data-linktext="Text link" data-href="#" data-class="dark-mode> </div>
