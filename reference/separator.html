Separators

You can include the following attributes when using the component. Note that those marked with * are compulsory:
  • data-include*: Name of html file of component
  • data-id*: ID of component when inserting to html
  • data-render: Set value to "true" if the component requires rendering. Most component requires rendering to set the class.
  • data-class*: Classes are used to render the different variants. Separate each class name with a space. Available class names are:
    • Horizontal separator: [Level 1] cmp-separator--section, [Level 2] cmp-separator--dotted, [Level 3] cmp-separator--solid--light, [Spacer] cmp-spacer
    • Vertical separator: [Dotted] cmp-separator-vertical--dotted, [Solid] cmp-separator-vertical--solid
Examples:

<!-- L1 Horizontal separator -->

<div data-include="horizontal_separator" data-id="hs_l1" data-render="true" data-class="cmp-separator--section"></div>

<!-- Dotted vertical separator -->

<div data-include="vertical_separator" data-id="hs_l1" data-render="true" data-class="cmp-separator-vertical--dotted"></div>
